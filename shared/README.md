# Shared Resources

This directory contains resources shared across multiple components of the IQ Tracker application.

## Contents

### API Contracts
- OpenAPI/Swagger specifications
- API endpoint documentation
- Request/response schemas

### Data Models
- Shared type definitions
- Database schema documentation
- Data validation rules

### Documentation
- Architecture diagrams
- System design documents
- Integration guides
- Data flow diagrams

### Constants
- Question types enumeration
- Difficulty levels
- Scoring constants
- Business logic constants (e.g., notification frequency)

## Usage

Components should reference shared schemas to ensure consistency across:
- iOS app API calls
- Backend API implementation
- Question service data models
